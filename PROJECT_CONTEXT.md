# Project Context: OOPs for Humans Academy

## Vision
To build the world's most intuitive, interactive learning platform for Object-Oriented Programming.
**Philosophy:**
- **Analogy First:** Explain concepts with real-world items (Lego, Pizza) before code.
- **Visuals over Text:** Use diagrams and interactive toys.
- **Play over Reading:** Users learn by doing.

## Core Features
1.  **Interactive Playgrounds:** "Toys" that visualize concepts (e.g., Cookie Factory for Classes).
2.  **Notebook Aesthetic:** A friendly, hand-drawn design style.
3.  **Multi-Language Support:** Java, Python, C++, TypeScript.

## Architecture
- **Framework:** Next.js 15 (App Router)
- **Content:** MDX (Markdown + React)
- **Styling:** Tailwind CSS + Framer Motion
